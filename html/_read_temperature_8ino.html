Detailed Description

Description

Example program for using the DS1631 library which allows access to the DS1631 I2C thermometer. The library as well as the most current version of this program is available at GitHub using the address https://github.com/SV-Zanshin/DS1631 and a more detailed description of this program (and the library) can be found at https://github.com/SV-Zanshin/DS1631/wiki/ReadTemperature.ino

The datasheet for the DS1631 can be found at https://datasheets.maximintegrated.com/en/ds/DS1631-DS1731.pdf
